🧩 Scientific Interpretation

  • UAT-CTGRU achieves the lowest prediction error and highest R², demonstrating strong explanatory power for quasar brightness variability.
  • The Transformer improves over Basic RNN via self-attention but lacks time-awareness, limiting its performance on irregularly sampled data.
  • Basic RNN fails to capture irregular temporal dependencies because it treats all time gaps identically.
  • The temporal attention mechanism provides interpretability — researchers can see which historical observations the model focuses on for each prediction.
  • Calibrated uncertainty (σ) from UAT-CTGRU enables reliable confidence intervals, critical for astronomical decision-making and follow-up observation planning.

🏗️ Model Architecture Overview

Baseline Basic RNN

Standard GRU (hidden=64). No attention mechanism. Treats all time gaps equally, losing long-range temporal context. Cannot model irregular sampling effects.

Baseline Transformer

2-layer TransformerEncoder (d=64, 4 heads). Self-attention across all tokens. Position-blind — does not account for actual time gaps between observations.

Proposed UAT-CTGRU

CTGRU encoder (hidden=128) with learnable time decay γ. Temporal attention weighted by time proximity. Produces calibrated uncertainty estimates (μ, σ).

🧬 CTGRU Time Decay Mechanism

The core innovation of the CTGRU encoder is the continuous-time decay:
h_decayed(t) = h(t-1) × exp(-γ × Δt)

  • γ (gamma) is a learnable parameter — the model learns the optimal decay rate
  • Δt is the actual time gap between consecutive observations
  • When Δt is small (frequent observations): decay ≈ 1, hidden state preserved
  • When Δt is large (long gap): decay → 0, hidden state fades

This elegantly handles the irregular sampling problem: the model's memory naturally decays during periods without observations, just as physical intuition would suggest — older measurements carry less information.

⚡ Attention Mechanism: Transformer vs UAT-CTGRU

Transformer Self-Attention

Computes attention between all pairs of tokens. Position information is via learned embeddings. It has no awareness of actual time differences. Every observation is equally accessible regardless of temporal distance.

UAT-CTGRU Temporal Attention

Attention is weighted by temporal proximity using a learnable time decay parameter. Recent observations get higher natural weight. Specifically designed for irregular time series.
